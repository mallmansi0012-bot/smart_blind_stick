# smart_blind_stick
This project is a smart stick designed to help visually impaired people navigate safely.  The stick detects obstacles and alerts the user through vibration and sound.
## Features
- Obstacle detection using ultrasonic sensor
- Vibration alert system
- Sound alert using buzzer
- GPS navigation for directions
- Emergency message to family members

## Components Used
- Arduino Uno
- Ultrasonic Sensor (HC-SR04)
- Buzzer
- Vibration Motor
- GPS Module
- GSM Module
- Battery

## Future Improvements
- Voice navigation
- Mobile app integration
- AI object recognition
